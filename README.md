# ☕ Java OOP Exercises

A collection of Java exercises focused on Object-Oriented Programming (OOP) concepts including encapsulation, inheritance, and abstraction.

## 📁 Project Structure

| File | Concepts Covered |
|------|-----------------|
| `BankAccount.java` | Encapsulation, getters, input validation |
| `Car.java` | Classes, attributes, methods |
| `Student.java` | Constructors, instance variables |
| `Vehicle.java` | Inheritance, superclass design |
| `Animal.java` | Polymorphism, method overriding |
| `Book.java` | Encapsulation, object modeling |
| `Rectangle.java` | Classes, mathematical methods |

## 🧠 Concepts Practiced

- **Encapsulation** — private fields with public getters/setters
- **Inheritance** — extending classes and reusing code
- **Polymorphism** — method overriding across subclasses
- **Abstraction** — modeling real-world objects in code
- **Constructors** — initializing objects properly

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Global-Wonder/Java-OOP-Exercises.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Java-OOP-Exercises
   ```

3. Compile a file:
   ```bash
   javac BankAccount.java
   ```

4. Run it:
   ```bash
   java BankAccount
   ```

## 🛠 Requirements

- Java JDK 8 or higher
- Any IDE (VS Code, IntelliJ, Eclipse) or terminal

## 📌 Notes

- Each `.java` file is a standalone exercise
- `.class` files are excluded via `.gitignore`
